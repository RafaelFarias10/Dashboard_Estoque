# Dashboard_Estoque
📊 Dashboard de Controle de Estoque e Vendas
Este projeto apresenta um dashboard interativo para análise de estoque, entradas, saídas e desempenho de vendas, com foco em tomada de decisão baseada em dados.

🚀 Visão Geral
O dashboard fornece uma visão consolidada dos principais indicadores do negócio:
📦 Total de produtos: 32
📥 Total de entradas: 2337
📤 Total de saídas: 2916
💰 Total de gastos: R$ 1,02M
🤑 Total vendido: R$ 4,31M

📈 Funcionalidades
Monitoramento de estoque em tempo real
Controle de entradas e saídas de produtos
Cálculo de saldo por item
Análise de custo total vs. venda total
Identificação de produtos com baixo estoque
Visualização clara e intuitiva para apoio estratégico

📉 Indicadores Importantes
Produtos com baixo estoque destacados para reposição rápida
Comparação entre custo e receita para análise de lucratividade
Controle detalhado por produto (entradas, saídas e saldo)

🛠️ Tecnologias Utilizadas
Power BI (ou ferramenta similar de visualização)
Excel / CSV para base de dados
Conceitos de análise de dados

🎯 Objetivo
Facilitar a análise de dados operacionais e financeiros, permitindo decisões mais rápidas e eficientes sobre estoque e vendas.
